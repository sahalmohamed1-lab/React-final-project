## Project overview
> Expense tracker is an app that helps a user keep track of his/her transactions by allowing them to add, update and delete a transaction.

## Installation
1. Clone the repository:
```bash
  git@github.com:sahalmohamed1-lab/React-final-project.git
```
2. Install dependencies:
```bash
npm install
```

## Usage
To run the project run the following command:
```bash
npm start
```
## Features
> A user is able to add an expense.
> A user is able to update an expense.
> A user is able to delete an expense.
> A user is able to see their total and highest transaction.
